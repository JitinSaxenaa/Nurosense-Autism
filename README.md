# Nurosense Autism Platform

Nurosense Autism is a premium AI-powered medical frontend dashboard designed for early detection and risk assessment of autism spectrum traits. It features a modern Single-Page Application (SPA) architecture, interactive CSS/Framer-Motion animations, and robust dynamic routing mimicking high-end clinical AI SaaS software.

## Features
- **Assessments:** Interactive question scales (Quick, Standard, and Full arrays).
- **Upload Portal:** Drag-and-drop document intake interfaces.
- **AI Dashboard:** A fully animated, glassmorphic analytics panel rendering Risk Levels, Confidence Scores, and textual Breakdowns.
- **Dynamic Fallbacks:** Built-in semantic randomization engine for simulation when the backend is offline.

## 🛠️ How to Run Locally

If you have cloned this repository, follow these quick steps to boot up the platform on your own computer:

### Prerequisites:
Make sure you have [Node.js](https://nodejs.org/) installed before starting.

### 1. Clone the repository
Download the project to your machine:
```bash
git clone https://github.com/JitinSaxenaa/Nurosense-Autism.git
```

### 2. Enter the project folder
Navigate into the directory:
```bash
cd Nurosense-Autism
```

### 3. Install dependencies
Install all the necessary packages and libraries (React, Tailwind CSS, Framer Motion, etc.):
```bash
npm install
```

### 4. Start the server
Spin up the Vite development server:
```bash
npm run dev
```

Once the server boots up, you will see a local URL in your terminal (usually `http://localhost:5173/`). Click it to launch the completely styled, animated web app!
